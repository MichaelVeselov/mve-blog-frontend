# MVE Blog - FrontEnd

## Portfolio App

### App Project (Frontend + Backend), using React + Vite

#### App features

- Registration and authorization
- Creating new posts (image + text)
- Creating comments
- Show all posts
- Show own posts
- Editing posts
- Deleting posts
- Deleting comments
